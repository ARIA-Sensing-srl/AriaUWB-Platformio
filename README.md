# AriaUWB-Platformio
PlatformIO extension for ARIA UWB modules

Toolchain:
Precompiled toolchain: https://drive.google.com/drive/folders/14DKVhOZQEVnwNfB2EFMPyWA22nspnS2o?usp=drive_link

Toolchain must be installed before trying to install the package.
Download the toolchain (zip)
open PlatformIO Core CLI and executes
pio pkg install -g --tool <path-to-toolchain-zip>

USB-JTAG adapter: C232HM-DDHSL-0
Under windows, cable driver must be replaced to libusb-win32
